# Neural Link × GBrain — 의도적 수집 및 투트랙 비즈니스 모델 (The Monetized Synthesis)

## 1. 한 줄 요약
> 무지성 자동 수집을 제거하고 **의도적 수집(Active Capture)**에 집중하며, **'유저 키 활용(BYOK)'**과 **'플랫폼 매니지드(마진율 100% 종량제)'** 투트랙으로 비용 방어와 수익화를 동시에 달성하는 독보적인 데이터 주권형 세컨 브레인 모델.

---

## 2. 비즈니스 모델: 투트랙 전략 (Two-Track Strategy)

초기 지식망 구축 및 유지에 드는 비용과 유저 진입 장벽을 완전히 분리하여, 파워 유저와 일반/기업 고객을 모두 흡수합니다.

### 트랙 A. 파워 유저용: 내 API 키 사용 (BYOK - Bring Your Own Key)
* **과금:** 플랫폼 이용료 무료 (유저가 소유한 OpenAI/Anthropic 등의 API 종량제 비용만 직접 부담)
* **메커니즘:** 클라이언트 앱 설정에 API 키를 입력하면, 서버를 거치지 않고 로컬 단에서 파싱 및 그래프 빌드 수행.
* **핵심 가치:** 비용에 민감하고 개발자 친화적인 유저들에게 완벽한 통제권과 프라이버시(민감 문서의 서버 유출 차단)를 제공하여 생태계 락인 유도.

### 트랙 B. 일반 및 엔터프라이즈용: 플랫폼 매니지드 서비스 (Managed Service)
* **과금:** **플랫폼 마진율 100%를 적용한 크레딧 충전식 종량제 (Pay-As-You-Go)**
* **메커니즘:** 유저가 크레딧을 충전하면, 플랫폼 서버에 연동된 고성능 xAI(Grok), GPT-4o 등의 API를 활용해 백엔드에서 대규모 파싱, 청크 분할, 에러 복구 등을 일괄 처리.
* **핵심 가치:** 복잡한 설정 없이 버튼 하나로 지식망을 구축해 주는 '원클릭 편의성'과, 깨진 JSON 구조를 서버 단에서 정밀 보정해 주는 '결과 보장형 최적화 파이프라인' 제공.

## 3. 종량제 가격 정책 및 수익성 시뮬레이션 (마진율 100% 기준)

* *산정 기준:* 1페이지 = 약 3,000자 (약 1,000 인풋 토큰), 추출되는 아웃풋 JSON = 약 200 토큰 내외.
* *매니지드 서비스 책정 구조:* **최종 유저 청구 금액 = API 원가 × 2.0 (마진율 100%)**

### [매니지드 서비스 트랙 책정 매트릭스]

#### 1) Claude 3.5 Sonnet 기반 처리 시 (인풋 $3/M, 아웃풋 $15/M)
* **100 페이지:** API 원가 $0.60 ➔ **유저 청구 금액 $1.20** (플랫폼 순수익 $0.60 / 약 800원)
* **500 페이지:** API 원가 $3.00 ➔ **유저 청구 금액 $6.00** (플랫폼 순수익 $3.00 / 약 4,000원)
* **1,000 페이지:** API 원가 $6.00 ➔ **유저 청구 금액 $12.00** (플랫폼 순수익 $6.00 / 약 8,000원)

#### 2) GPT-4o 기반 처리 시 (인풋 $5/M, 아웃풋 $15/M)
* **100 페이지:** API 원가 $0.80 ➔ **유저 청구 금액 $1.60** (플랫폼 순수익 $0.80 / 약 1,050원)
* **500 페이지:** API 원가 $4.00 ➔ **유저 청구 금액 $8.00** (플랫폼 순수익 $4.00 / 약 5,300원)
* **1,000 페이지:** API 원가 $8.00 ➔ **유저 청구 금액 $16.00** (플랫폼 순수익 $16.00 / 약 10,600원)

> **전략적 포지셔닝:** 유저 입장에서는 평생 모은 귀중한 문서 1,000장을 완벽한 AI 지식망으로 바꾸는 데 단돈 15,000원 상당($12~$16)을 지출하는 것이므로 심리적 저항감이 매우 낮습니다. 반면 플랫폼은 고정비 리스크 없이 고스란히 원가만큼의 순수익(마진율 100%)을 확보하게 됩니다.

## 4. 시장 차별화 포인트: 안티 블랙박스 (Anti-Blackbox)와 MCP

타사의 닫힌 AI 메모리 기능과 차별화되는 본 모델의 독보적인 세일즈 프로포지션(USP)입니다.

1. **완전한 데이터 주권 (Data Sovereignty):** 유저가 구축한 세컨 브레인 지식망은 특정 기업의 서버 DB에 종속되지 않고, 오직 유저 자신의 GitHub 레포지토리에 투명한 마크다운과 구조화된 `.cotext/neural.json` 파일 형태로 영구 보존됩니다.
2. **무한한 확장성 (MCP Connectivity):** 이 지식망은 단순한 노트 앱 내부용이 아닙니다. 플랫폼이 제공하는 **로컬/원격 MCP(Model Context Protocol) 서버**를 통과하므로, 유저는 Cursor, Claude Desktop, ChatGPT 등 자신이 원하는 시장의 그 어떤 최신 AI 에인전트와도 이 두뇌를 즉시 연결하여 자유롭게 활용할 수 있습니다.

## 5. 핵심 기술 구현 요약 (Fail-Safe)

* **의도적 수집(Active Capture) 어시스트:** 사용자가 PC나 모바일에서 채팅하듯 메모를 남기면, 백그라운드에서 가벼운 모델이 작동하여 추천 클러스터와 관련 노드를 UI 칩으로 제안, 탭 한 번으로 분류를 완료합니다.
* **청크 분할 제어 (Chunked Relay):** LLM의 출력 토큰 한계(4k~8k)로 파싱이 끊기는 문제를 방지하기 위해 클라이언트/서버 단에서 문서를 5~10페이지 단위로 쪼개어 순차 호출하는 릴레이 파싱 로직을 적용합니다.
* **JSON 복구 엔진 (JSON Repair Engine):** 매니지드 서비스 트랙에서는 LLM의 JSON 반환 오류(괄호 누락 등)를 백엔드에서 유효성 검증(Zod) 및 정규식 기반으로 자동 보정하여 완벽한 정본을 보장합니다.

## 6. 최종 명제
> **"편리함을 원하는 유저에게는 원클릭의 가치를 팔아 원가만큼의 순수익을 남기고, 통제권을 원하는 파워 유저에게는 무료 프리미엄을 주어 생태계를 장악한다. 축적된 두뇌는 MCP를 통해 유저의 영원한 자산이 된다."**
